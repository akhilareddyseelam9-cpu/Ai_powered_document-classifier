# 📄 AI Document Classifier & Summarizer

## 🚀 Project Overview
This is an AI-powered Streamlit web app that:
- Classifies documents
- Generates smart summaries
- Shows category distribution (Pie + Bar charts)
- Supports Login / Register system
- Stores user history using SQLite

---

## ✨ Features

✔ User Authentication (Login/Register/Forgot Password)  
✔ AI Document Classification  
✔ Smart Text Summarization  
✔ Upload PDF / DOCX / TXT  
✔ Pie Chart + Bar Chart Visualization  
✔ Download Report (TXT)  
✔ User-based History Tracking  
✔ SQLite Database Integration  

---

## 🛠️ Tech Stack

- Python 🐍
- Streamlit 🎈
- SQLite 🗄️
- PyPDF2 📄
- docx2txt 📑
- Matplotlib 📊
